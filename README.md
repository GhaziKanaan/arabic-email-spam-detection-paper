# Arabic Email Spam Detection Using Machine Learning and Deep Learning: A Comparative Study

## Overview

This repository contains the published research paper:

**Arabic Email Spam Detection Using Machine Learning and Deep Learning: A Comparative Study**

The study investigates the effectiveness of machine learning and deep learning approaches for Arabic email spam detection, addressing the unique linguistic challenges of Arabic Natural Language Processing (NLP).

---

## Research Contributions

* Developed a comparative framework for Arabic email spam detection.
* Evaluated both Machine Learning and Deep Learning approaches.
* Applied Arabic-specific text preprocessing techniques.
* Compared SVM, Naïve Bayes, Random Forest, and LSTM models.
* Demonstrated the effectiveness of machine learning methods for Arabic spam classification.

---

## Models Evaluated

* Support Vector Machine (SVM)
* Naïve Bayes
* Random Forest
* Long Short-Term Memory (LSTM)

---

## Key Results

| Model         | Accuracy | Precision | Recall | F1 Score |
| ------------- | -------- | --------- | ------ | -------- |
| SVM           | 99%      | 100%      | 98%    | 99%      |
| LSTM          | 96%      | 99%       | 97%    | 98%      |
| Random Forest | 94%      | 94%       | 88%    | 91%      |
| Naïve Bayes   | 94%      | 89%       | 96%    | 92%      |

The Support Vector Machine (SVM) achieved the best overall performance, reaching 99% accuracy and a 99% F1-score.
